## WebRTC는 어떤 기술인가요?
WebRTC는 웹 브라우저나 모바일 앱 간에 별도의 플러그인 없이 실시간 미디어(음성, 영상) 및 데이터 전송을 가능하게 하는 오픈소스 프로젝트이자 표준 기술입니다. 

구현방법으로는 대표적으로 P2P, SFU, MCU 방식이 있습니다. 각 방식은 서버의 개입 정도와 클라이언트의 부하 사이에서 트레이드오프 관계가 있습니다.

| **구분**       | **Mesh (P2P)**   | **SFU (Forwarding)** | **MCU (Mixing)**    |
| ------------ | ---------------- | -------------------- | ------------------- |
| **특징**       | 서버 없이 단말끼리 직접 연결 | 서버가 미디어 스트림을 **중계**  | 서버가 미디어를 **하나로 합침** |
| **클라이언트 부하** | 높음 (참여자가 늘수록 급증) | 중간                   | 낮음                  |
| **서버 비용**    | 거의 없음 (시그널링만 수행) | 중간                   | 매우 높음               |
| **지연 시간**    | 가장 낮음            | 낮음                   | 높음 (인코딩 과정 필요)      |
